# 2% Realty Regina Broker Landing Page

A responsive, accessible single-page website for a Regina real estate professional.

## Preview

Run a local static server from this directory, for example:

```powershell
python -m http.server 4173
```

Then open `http://localhost:4173`.

## Before launch

- Replace `YOUR CLIENT'S NAME`, phone number, email address, and footer text in `index.html`.
- Add the approved client portrait and brokerage logo.
- Add an approved Regina neighbourhood photo as `assets/regina-neighbourhood.jpg`.
- Replace the placeholder testimonial with a verified review or remove the section.
- Follow `google-apps-script/SETUP.md` to connect the inquiry form.
- Confirm official brokerage branding, legal disclosures, and trademark usage with the client.
